# Artifacts & Reproducibility Index

Every result is backed by a committed pre-registration (before the run), a raw
result JSON (per-episode tool-call traces), a results doc, and a git tag. All
reported figures are regenerated by `scripts/audit_phase3a.py`.

Environment: Python 3.14.6, `requirements.lock` (openai 2.44.0, anthropic
0.112.0, pydantic 2.13.4, typer 0.26.7). Offline harness + 121 tests; deterministic
mock for everything except the real-model runs below.

## Exact commands (real-model runs)

```bash
# clean confirmatory (CANONICAL)
pplab agentic --client openai --model gpt-4.1 --temperature 0.3 \
  --scenario-set confirmatory --reps 5 --seed 20260626 \
  --sleep-between-episodes 0.4 --output reports/phase3a_clean_confirmatory_gpt41_t03.json

# temperature robustness
pplab agentic --client openai --model gpt-4.1 --temperature 0.0 --scenario-set confirmatory --reps 3 --seed 20260626 --output reports/phase3a_robust_gpt41_t00.json
pplab agentic --client openai --model gpt-4.1 --temperature 0.7 --scenario-set confirmatory --reps 3 --seed 20260626 --output reports/phase3a_robust_gpt41_t07.json

# external validity (Claude); driver smoke first (reps 1)
pplab agentic --client anthropic --model claude-sonnet-4-5 --temperature 0.3 --scenario-set confirmatory --reps 1 --seed 20260626 --output reports/phase3a_claude_smoke.json
pplab agentic --client anthropic --model claude-sonnet-4-5 --temperature 0.3 --scenario-set confirmatory --reps 5 --seed 20260626 --output reports/phase3a_claude_external_t03.json

# regenerate every reported figure from raw traces
.venv/bin/python scripts/audit_phase3a.py
```

API keys are read from the environment (`OPENAI_API_KEY` / `ANTHROPIC_API_KEY`)
and never written to any artifact (manifests store the model name only; error
text is credential-scrubbed).

## Audit invariants (verified by `scripts/audit_phase3a.py`)

- 0 `authorization_enforcement_failures` in every condition.
- Unauthorized **execution** rate = 0 on every unauthorized arm (gateway), reported
  separately from the **attempt** rate.
- Positive control execution = 1.00; negative (scope-mismatch) control = 0.00.
